## Lessons learned

- KPI definitions must be agreed before dashboard build begins.
- Counselors need action-oriented screens, not executive-style dashboards.
- Principals need exception lists behind KPI tiles to trust the metrics.
- Data governance language must clarify that analytics are operational, not diagnostic.
