# Product Analytics

## Product usage metrics

| Metric | Definition | Product question |
|---|---|---|
| Daily Active Users | Unique users with activity per day | Are staff using SafePulse daily? |
| Monthly Active Users | Unique users with activity per month | Is adoption expanding? |
| Feature Adoption | Users using a feature / eligible users | Which workflows are gaining traction? |
| Session Duration | Time from login to inactivity | Is the experience efficient? |
| Intake Funnel Completion | Submitted requests / started intake forms | Are users dropping off during intake? |
| Notification Effectiveness | Notifications followed by action / notifications sent | Are reminders driving workflow completion? |
| Search Usage | Searches per active user | Are users relying on search to locate cases? |
| Report Usage | Dashboard/report views per role | Which reports are valuable? |
| Drop-off Analysis | Exit step by workflow | Which screens create friction? |

## Funnel example

1. User opens intake page
2. Selects school and category
3. Completes required details
4. Submits request
5. Case created successfully
